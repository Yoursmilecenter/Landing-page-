# Smile Center - Short Invite Links Guide

## Problem Solved
**Before:** Long, unprofessional links
```
https://yoursmilecenter.github.io/Landing-page-/portal-login.html?invite=nyrkx602mi219z9s&phone=0547609644
```

**After:** Clean, professional short links
```
https://yoursmilecenter.github.io/join?c=nyrkx602&p=0547609644
```

---

## How to Use

### Option 1: Ultra-Short (Recommended)
**URL Format:**
```
https://yoursmilecenter.github.io/join?c=INVITECODE&p=PHONE
```

**Example:**
```
https://yoursmilecenter.github.io/join?c=nyrkx602&p=0547609644
```

**URL Parameters:**
- `c=` or `code=` or `i=` - Invite code (shortened to just first 8 chars)
- `p=` or `phone=` - Phone number

---

### Option 2: Slightly Longer
**URL Format:**
```
https://yoursmilecenter.github.io/invite?i=INVITECODE&p=PHONE
```

**Example:**
```
https://yoursmilecenter.github.io/invite?i=nyrkx602&p=0547609644
```

---

## Benefits

✅ **66% Shorter** - Much easier to type and share
✅ **Professional** - Clean, branded appearance
✅ **Flexible** - Supports multiple parameter formats
✅ **Mobile-Friendly** - Easier to click and share on phones
✅ **SMS-Friendly** - Won't break in text messages

---

## Technical Details

Both `join.html` and `invite.html` are redirect pages that:
1. Show your logo with animation
2. Display "Redirecting..." message
3. Automatically forward to portal-login.html with full parameters
4. Take less than 1 second to redirect

---

## When to Use Each

**Use `join`** - For new client invites (shortest, most professional)
**Use `invite`** - Alternative if you prefer the word "invite" in URL

Both work exactly the same way!
