### 3. Title (0.3s)

- Slides up 20px with fade-in

### 4. Subtitle (0.5s)
- "Experience revolutionary dental design..."

### 5. Feature Cards (0.7s)

- Smooth easing animation

## ✨ Special Effects

### Logo Glow Animation
```css
0%   → shadow: 8px blur, 40% opacity
50%  → shadow: 16px blur, 80% opacity
100% → shadow: 8px blur, 40% opacity
```
**Duration:** 3 seconds, infinite loop

### Background Rotation
```css
Gradient orb rotates 360° every 20 seconds
Creates subtle movement without distraction
```

### Hover Effects
```css
Cards lift 4px on hover
Box shadow appears: 8px blur with blue glow
Background lightens slightly
All transitions: 0.3s ease
```
